jgraphUI is a rich feature and highly customizable yet easy to use diagram tool, this manual will list few samples and a detailed parameter list for graph and node elements.

Baisc example

The output of this sample code will be a diagram area with 1000px width and 600px height, positioned at 20px, 30px.

var diagram = new Diagram({
        'xPosition':20,
        'yPosition':30,
        'width':1000,
        'height':600,
        'imagesPath': 'images/',
        'connectionColor': '#AA0000'
      });

Override save method

The output of this sample code will be the same as the previous sample, added override to onSave event.

var diagram = new Diagram({
        'xPosition':20,
        'yPosition':30,
        'width':1000,
        'height':600,
        'imagesPath': 'images/',
        'connectionColor': '#AA0000',
        onSave: function(data){
          alert('from on save event \n' +data);
        }
      });

Customize toolbar buttons

The output of this sample code will be the same as the previous sample, but the toolbar will show only three buttons (Add, Save and Delete).

var diagram = new Diagram({
        'xPosition':20,
        'yPosition':30,
        'width':1000,
        'height':600,
        'toolbar_add_button' : true,
        'toolbar_save_button' : true,
        'toolbar_delete_button' : true,
        'toolbar_background_color_button' : false,
        'toolbar_border_color_button' : false,
        'toolbar_font_color_button' : false,
        'toolbar_font_size_button' : false,
        'toolbar_font_family_button' : false,
        'toolbar_border_width_button' : false,
        'connectionColor': '#AA0000',
        'imagesPath': 'images/',
        onSave: function(data){
          alert('from on save event \n' +data);
        }
      });


Parameter list for graph object

Parameter Description Default Value
id unique id for usage in code if not provided, unique value will be auto generated
xPosition left graph position 20
yPosition top graph position 30
width graph width 1000
height graph height 500
connectionWidth thickness of the connection line 3
connectionColor color of the connection line #888888
imagesPath where to find graph images images/
noToolbar if set to true, the graph will have no toolbar false
toolbar_add_button if set to false, no "Add Node" button will appear on the toolbar true
toolbar_save_button if set to false, no "Save" button will appear on the toolbar true
toolbar_delete_button if set to false, no "Delete" button will appear on the toolbar true
toolbar_background_color_button if set to false, no "Background Color" button will appear on the toolbar true
toolbar_border_color_button if set to false, no "Border Color" button will appear on the toolbar true
toolbar_font_color_button if set to false, no "Font Color" button will appear on the toolbar true
toolbar_font_size_button if set to false, no "Font Size" button will appear on the toolbar true
toolbar_font_family_button if set to false, no "Font Family" button will appear on the toolbar true
toolbar_border_width_button if set to false, no "Border Width" button will appear on the toolbar true
onSave function decleration to define what to do when "Save" button is clicked, you can show save indicator here do Nothing


Adding nodes to the diagram using code is simple, following code sample will add a node element to the diagram object created in the previous three samples, in this sample, all the node parameters are provided, but you may not provide all of them at real time project, you can customize the code upon requirements.

diagram.addNode(new Node({
        'nodeId': '1',
        'nodeType':'NODE',
        'nodeContent': 'Node Content',
        'xPosition':406,
        'yPosition':250,
        'width': '120',
        'height' : '70',
        'bgColor':'#FFFFFF',
        'borderColor':'#AAAAAA',
        'borderWidth':'1',
        'fontColor':'#000000',
        'fontSize':'',
        'fontType':'',
        'minHeight':50,
        'maxHeight':200,
        'minWidth':100,
        'maxWidth':200,
        'nPort':true,
        'ePort':true,
        'sPort':true,
        'wPort':true,
        'image':'',
        'draggable':true,
        'resizable':true,
        'editable':true,
        'selectable':true,
        'deletable':true,
        'nPortMakeConnection': true,
        'ePortMakeConnection': true,
        'sPortMakeConnection': true,
        'wPortMakeConnection': true,
        'nPortAcceptConnection': true,
        'ePortAcceptConnection': true,
        'sPortAcceptConnection': true,
        'wPortAcceptConnection': true
      }));

Parameter list for node element

Parameter Description
nodeId unique id for usage in code, if not provided, unique value will be auto generated
nodeType NODE, TEXT or IMAGE -
nodeContent text inside node -
xPosition left node position
yPosition top node position
width node width
height node height
bgColor node background color
borderColor node border color
borderWidth node border thickness
fontColor node content font color
fontSize node content font size
fontType node content font type
minHeight minimum node height when resize
maxHeight maximum node height when resize
minWidth minimum node width when resize
maxWidth minimum node width when resize
nPort does the node have north port (true, false)
ePort does the node have east port (true, false)
sPort does the node have south port (true, false)
wPort does the node have west port (true, false)
nPortMakeConnection does the node make connections from the north port (true, false)
ePortMakeConnection does the node make connections from the east port (true, false)
sPortMakeConnection does the node make connections from the south port (true, false)
wPortMakeConnection does the node make connections from the west port (true, false)
nPortAcceptConnection does the node accept connections to the north port (true, false)
ePortAcceptConnection does the node accept connections to the east port (true, false)
sPortAcceptConnection does the node accept connections to the south port (true, false)
wPortAcceptConnection does the node accept connections to the west port (true, false)
image image file name and path, required if node type is IMAGE
imageWidth image width, required if node type is IMAGE
imageHeight image height, required if node type is IMAGE
draggable is node draggable (true, false)
resizable is node resizable (true, false)
editable is node content editable (true, false)
selectable is node draggable (true, false)
deletable is node draggable (true, false)


Adding a connection line between node elements can be done using one line of code, the following sample will draw a connection line from the south port on source node to the north port of the target node, connection line will be dark red with 3px width, the method declaration is:
diagram-object.addConnection(new Connection('source node id', 'source node port [n,e,s,w]', 'target node id', 'target node port [n,e,s,w]', 'color', 'width'))


diagram.addConnection(new Connection('1','s','2','n','#AA0000','3'));
